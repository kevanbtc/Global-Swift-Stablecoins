## Reporting a Vulnerability

We take the security of Unykorn Layer 1 seriously. If you believe you have found a security vulnerability, please report it to us as described below.

### How to Report a Vulnerability

Please send an email to security@unykorn.io with:

1. Description of the vulnerability
2. Steps to reproduce the issue
3. Potential impact of the vulnerability
4. Any mitigations you've identified

### What to Expect

- Within 24 hours: Acknowledgment of your report
- Within 72 hours: Initial assessment of the vulnerability
- Within 7 days: Detailed response with next steps
- Within 90 days: Public disclosure (if applicable)

### Responsible Disclosure

We kindly ask you to:

- Not disclose the vulnerability to others
- Give us reasonable time to respond
- Not exploit the vulnerability
- Provide enough information to reproduce the issue
