### Installation

```bash

# Install dependencies

# Copy environment configuration (create .env from template if it doesn't exist)
```

### Initial Setup & Deployment

#### 1. Start Local Development Node

```bash
# Start a local Hardhat node
npx hardhat node

# In a new terminal, deploy contracts
npm run deploy:local
```

#### 2. Configure Compliance Framework

Set up your compliance policies and KYC profiles:

```typescript
// Set up a compliance policy
await registry.setPolicy(policyId, {
    allowUS: true,          // Allow US investors
    allowEU: true,          // Allow EU investors  
    regD506c: true,         // Regulation D 506(c) compliant
    micaART: true,          // MiCA Asset-Referenced Token
    proOnly: false,         // Open to retail investors
    travelRuleRequired: true // FATF Travel Rule compliance
});

// Register and verify a user's KYC
await registry.setProfile(userAddress, {
    kyc: true,              // KYC verified
    accredited: true,       // Accredited investor status
    kycAsOf: Date.now(),    // KYC verification timestamp
    kycExpiry: futureDate,  // KYC expiration date
    isoCountry: "US",       // User's jurisdiction
    frozen: false           // Account status
});
```

#### 3. Deploy Your First Stablecoin

```bash
# Deploy a compliant stablecoin
npx hardhat run scripts/deploy-stablecoin.js --network localhost

# Mint initial supply (admin only)
npm run mint -- --amount 1000000 --to 0xYourAddress
```

### Example Usage

#### Transfer with Compliance Checks

```typescript
// Transfer tokens (compliance checks happen automatically)
const tx = await stablecoin.transfer(recipientAddress, amount);
await tx.wait();
console.log("Transfer successful with compliance validation!");
```

#### Check Compliance Status

```typescript
// Verify if a transfer would be compliant before executing

    // Handle compliance failure
```

```typescript

```
