# Validator Onboarding Guide

## Overview

This document outlines the process for onboarding new validators to the Unykorn Layer 1 network.

## Requirements

### Hardware Requirements

- CPU: 32 cores
- RAM: 64GB
- Storage: 2TB NVMe SSD
- Network: 1Gbps dedicated connection
- Backup Power Supply

### Software Requirements

- Operating System: Ubuntu 20.04 LTS
- Node Software: Besu v1.5.3
- Security Suite: Advanced Intrusion Detection
- Monitoring Tools: Prometheus + Grafana
